SaaS-Pro is a vertical SaaS MVP targeting artisans/niche professions. First vertical: ramoneurs (chimney sweeps).

**Why:** Build a reusable SaaS platform with trade-specific modules, starting with chimney sweeps.

**How to apply:**
- Common modules: Auth, Dashboard (revenue/clients/appointments stats), Client management (customizable fields per trade), Calendar/planning, Basic invoicing (quotes + invoices as PDF)
- Chimney sweep layer: chimney type, last visit, chimney sweep certificate PDF generation, annual email reminders
- Stack: Next.js 16 + Tailwind v4 + PostgreSQL on Railway (yamanote.proxy.rlwy.net:27751)
- Design: modern, clean, Stripe-inspired
- Language: French UI
