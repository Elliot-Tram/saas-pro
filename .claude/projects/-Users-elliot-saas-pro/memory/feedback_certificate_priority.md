Le certificat de ramonage est le coeur de l'outil - c'est la feature n°1.

**Why:** C'est le document officiel que le ramoneur remet au client, requis par la loi et les assurances. Si ce document est top, l'outil se vend.

**How to apply:**
- Le certificat PDF doit être personnalisable : logo entreprise, couleurs, mise en page pro
- Signature électronique intégrée (tactile sur téléphone/iPad)
- Le document final doit avoir un rendu "officiel" irréprochable
- Priorité absolue sur le polish de ce module avant tout le reste
