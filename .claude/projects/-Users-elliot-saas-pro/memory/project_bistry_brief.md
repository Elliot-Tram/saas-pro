## Branding
- Nom: Bistry (du bistre, dépôt dans les conduits)
- Domaine: bistry.fr
- Cible: ramoneurs indépendants et petites équipes (2-5 personnes)
- Tagline: "Votre logiciel actuel vous aide à gérer vos clients. Bistry vous aide à en avoir plus."

## Pricing
- Essentiel 29€/mois: 1 user, fiches clients, planning, certificats PDF, facturation/devis, rappels, dashboard
- Pro 49€/mois: + site vitrine template, demande avis Google auto, stats avancées
- Boost 79€/mois: + publication photos GMB, setup GMB inclus
- Siège supplémentaire: +19€/mois par user (tous paliers)

## Killer features à builder
1. Workflow post-intervention: clic "terminé" → certificat PDF auto → demande avis Google → photo stockée pour GMB
2. Site vitrine (Pro): template dupliqué par client, formulaire devis → fiche client auto, déployé en 30 min
3. Migration données: import CSV ou saisie depuis photos de carnet

## Design principles
- Mobile-first (terrain, pas PC)
- Gros boutons, parcours évident, zéro complexité
- L'opposé des logiciels métier années 2000

**How to apply:** Toutes les décisions produit/design doivent être filtrées par ces principes. Le workflow post-intervention est la feature prioritaire.
