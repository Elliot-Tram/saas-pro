Le PDF du certificat est fonctionnel mais ne fait pas "waouh". Il manque le côté premium/design qui fait dire "c'est beau".

**Why:** C'est LE document que le client garde. Si le ramoneur est fier de le montrer, l'outil se vend tout seul.

**How to apply:** Retravailler le PDF avec une approche plus design : peut-être utiliser une librairie PDF plus puissante (Puppeteer/HTML-to-PDF) pour avoir plus de contrôle sur la mise en page, ou investir dans un template HTML converti en PDF. pdf-lib est limité visuellement.
