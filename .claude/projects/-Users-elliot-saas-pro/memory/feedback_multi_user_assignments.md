---
name: Multi-user assignment for appointments
description: User wants to prepare for multi-employee companies where appointments can be assigned to specific team members
type: feedback
---

Prévoir l'attribution des RDV à un membre de l'équipe. Même si pour l'instant c'est mono-utilisateur, afficher "Assigné à: [nom]" sur les RDV dans "Ma journée" pour anticiper le multi-utilisateur.

**Why:** Les entreprises de ramonage ont souvent 2-3 personnes. Il faut pouvoir dispatcher les RDV.

**How to apply:** Pour le MVP, afficher le nom de l'utilisateur assigné. Plus tard, ajouter un modèle TeamMember et un champ assignedTo sur Appointment.
