---
name: Logout should be in top-right user menu
description: User finds logout at bottom of sidebar unintuitive. Prefers it in a top-right profile dropdown like most SaaS apps.
type: feedback
---

La déconnexion en bas de la sidebar n'est pas instinctive. L'utilisateur s'attend à la trouver en haut à droite, dans un menu profil (clic sur avatar/photo).

**Why:** Convention UX standard dans les SaaS (Stripe, Notion, etc.) — le logout est dans un dropdown du profil en haut à droite.

**How to apply:** Déplacer la déconnexion dans un menu dropdown sur l'avatar dans le header. Le header affiche déjà l'initiale de l'utilisateur en haut à droite.
